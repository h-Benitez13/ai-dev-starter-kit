# Selling Your Own Starter Kits

This guide covers how to monetize AI dev starter kits and similar digital products.

## Platforms

### Gumroad (Recommended for starting)

- **URL**: https://gumroad.com
- **Fee**: 10% per sale
- **Pros**: Simple setup, handles payments, built-in audience discovery
- **Best for**: Digital downloads, one-time purchases

### Lemon Squeezy

- **URL**: https://lemonsqueezy.com
- **Fee**: 5% + $0.50 per sale
- **Pros**: Lower fees, handles taxes globally, clean checkout
- **Best for**: Digital products with international customers

### Your Own Site

- **Stack**: Stripe + a static site
- **Fee**: Stripe's 2.9% + $0.30
- **Pros**: Full control, lowest fees, own the customer relationship
- **Best for**: Once you have consistent sales

## Pricing Strategy

### For AI Dev Starter Kits

| Tier        | Price | What's Included                                   |
| ----------- | ----- | ------------------------------------------------- |
| **Free**    | $0    | 2-3 basic skills (open source on GitHub)          |
| **Starter** | $19   | All skills + Cursor rules + setup guide           |
| **Pro**     | $39   | Starter + customization guide + video walkthrough |
| **Team**    | $79   | Pro + team license + support                      |

### Why This Works

- **Free tier** builds trust and drives traffic (your GitHub repo)
- **Paid tiers** add convenience (customization, video, support)
- People pay for curation and time-saving, not raw content

## Marketing Channels

### Twitter/X (Highest ROI for dev tools)

1. Share short clips of your AI workflow (screen recordings)
2. Post the "before vs after" of using your skills
3. Thread format: "How I save 2 hours/day with AI coding tools" -> link to product
4. Engage with AI coding tool communities

### Dev Communities

- **Reddit**: r/cursor, r/ClaudeAI, r/ChatGPTCoding
- **Discord**: Cursor Discord, Claude Discord, OpenCode communities
- **Hacker News**: Show HN posts get massive visibility
- **Dev.to / Hashnode**: Write tutorials that naturally lead to your product

### GitHub

- Keep the free tier public on GitHub
- Stars = social proof = sales for the paid tier
- Add a "Get the Pro version" link in your README

## Content Funnel

```
GitHub repo (free skills) -> social proof (stars)
     |
Twitter threads showing workflow -> engagement
     |
Blog post / tutorial -> SEO traffic
     |
Product page (Gumroad) -> sales
     |
Email list -> repeat customers for future products
```

## Expanding Your Product Line

Once you have a selling AI dev starter kit, you can expand:

1. **Stack-specific kits**: "Next.js AI Dev Kit", "Python AI Dev Kit"
2. **Role-specific kits**: "Frontend Lead Kit", "DevOps AI Kit"
3. **Template packs**: shadcn dashboard templates, component libraries
4. **Video courses**: "Production AI Dev Workflows" (higher price point)
5. **Consulting**: 1-on-1 setup for teams ($200-500/session)

## Revenue Expectations

Be realistic:

| Timeline  | Expected Revenue | What's Happening                   |
| --------- | ---------------- | ---------------------------------- |
| Month 1   | $0-50            | Building product, first launch     |
| Month 2-3 | $50-200          | Growing awareness, iterating       |
| Month 6   | $200-500         | Consistent sales, repeat content   |
| Month 12  | $500-2000        | Multiple products, email list, SEO |

These are conservative estimates for a side project. The key is consistency, not virality.

## Legal Notes

- **MIT License** on the free tier means anyone can use/modify it
- For paid tiers, you can use a commercial license
- Don't include any proprietary code from your employer
- Generic patterns and workflows are fine to sell -- they're your knowledge
